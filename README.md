Rental Management App — это полнофункциональное desktop-приложение для управления арендой недвижимости, разработанное на Python с использованием Tkinter. Приложение позволяет эффективно управлять объектами недвижимости, арендаторами, договорами аренды и платежами, а также формировать различные отчеты.

🚀 Возможности
Основные функции

📊 Дашборд-Централизованный просмотр статистики: количество объектов, свободные/арендованные помещения, выручка за месяц, предстоящие события

🏠 Недвижимость-Добавление, редактирование, удаление объектов; экспорт списка в CSV

👥 Арендаторы-Управление базой арендаторов с контактной информацией

📄 Договоры аренды-Создание и управление договорами, продление аренды, контроль сроков

💰 Платежи-Учет платежей, фильтрация по месяцам/годам

📈 Отчеты-Финансовые отчеты, отчеты по арендаторам и недвижимости, экспорт данных

⚙️ Настройки-Настройка валюты, резервное копирование, восстановление БД

Технические особенности

✅ Полноценная работа с SQLite базой данных

✅ Автоматическое создание таблиц и тестовых данных

✅ Поддержка CSV и DOCX экспорта отчетов

✅ Резервное копирование и восстановление данных

✅ Интуитивный интерфейс с цветовой индикацией статусов

✅ Валидация ввода данных
